# nest-pokedex

## Local Running
1. Git clone this repo
2. Execute the following commands

```bash
pnpm install
```

3. Run docker compose for mongodb

```bash
docker-compose up -d
```

4. Run the application

```bash
pnpm run start:dev
```

5. Exectue the seed

```bash
http://localhost:3000/seed
```
